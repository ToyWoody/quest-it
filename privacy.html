HECAS 이 앱은 "android.permission.READ_PHONE_STATE" 권한을 요구합니다. This app requires "android.permission.READ_PHONE_STATE" 이유 : 이 앱의 기능 중 휴대폰을 켰을 때 바로 화면에 어휘를 표시해주는 기능이 있습니다. Reason : This app has a functionality showing vocabulary using screen lock. 그런데 전화를 사용하는 중에는 잠금 화면이 켜지지 않도록 하기 위해 해당 정보를 활용합니다. However, in case of while on calling, screen lock should not be activated. So app use phone state to check it. phone state - 전화중인지 여부를 확인 phone state - information whether calling or not.
